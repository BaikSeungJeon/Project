기술 블로그 작성

개발 공부를 처음 접하면서, 그날의 복습을 위해 기술 블로그를 시작했습니다.

처음부터 저의 발전을 남기고 싶어 시작한 것도 있지만,
개발 분야 공부는 작은 기록이라도 나비 효과처럼 큰 결과를 낸다고 믿었기 때문입니다.

실제로 HTML, CSS 그리고 React와 깃, 깃허브 사용법,
더 나아가 서버를 다루는 nodeJs 공부를 할 때도 전부 다 기록을 하였습니다.

사용 기술

제가 블로그에 정리한 복습 내용과, 저의 발자취를 구경하러 오세요.

언더아머 홈페이지 클론코딩

HTML, CSS, JS를 입문하고,
처음으로 실제 웹 사이트를 구현한 작업입니다.

대기업 쇼핑몰의 홈페이지를 클론코딩 함으로써,
보편적인 웹 사이트인 만큼 기본적인 내용과 구조,
쇼핑몰의 상품 이미지와 로고 배치 등의 감각을 익힐 수 있었습니다.

사용 기술

클론코딩 한 언더아머에서 의류를 구매해 보세요.

깃허브 공부하기

개발자에게 있어 깃과 깃허브는 필수라고 생각합니다.
오히려 잘하는 게 당연한 게 아닌, 못하면 되려 안 된다 생각했을 정도입니다.

직업 특성 상 코드를 작성하는 능력도 중요하지만,
팀 내에 의사소통과 협업 능력은 그 이상으로 중요하다 생각했습니다.

따라서 현재 진항 상황을 파악하고 공유할 수 있는 깃허브를 공부하고,
매일 같이 저의 작업물을 커밋, 푸쉬할 수 있도록 목표를 세워 공부했습니다.

사용 기술

초보 개발자의 깃허브가 궁금하신 분들께선 아이콘을 클릭해 주세요.

반응형 홈페이지

모바일 화면과 PC 화면 등,
다양하게 변화하는 반응형 홈페이지를 만들었습니다.

코로나 19로 비대면 서비스가 가속화 되어 가는 가운데,
본인의 사진을 올리는 '비대면 전시회' 서비스 성격의 홈페이지입니다.

사용 기술

저의 갤러리에 초대합니다. 제 사진을 감상해 주세요.

자바스크립트 취약한 기능 복습

프론트를 작업하는 웹 개발자에게 있어 자바스크립트는 매우 중요한 기술이라 생각합니다.

그 중, 반응형 모바일 화면에서 클릭 시 Nav Bar가 보이는 동작과,
투두 리스트를 포함한 리스트 작성은 많이 쓰이면서도 중요하다고 생각했습니다.

당시엔 jQuery만 사용해 작업을 하느냐 순수 자바스크립트의 감을 살짝 잃었던 상태였고,
더 늦게 전에 기본기부터 다시 잡고 싶어 연습한 작업입니다.

자바스크립트를 연습하기 위해 화면이 필요했기 때문에,
덕분에 HTML5와 CSS3을 이용한 마크업까지 복습할 수 있는 시간이었습니다.

리액트로 키움증권 MTS 클론코딩

리액트로 '쇼핑몰 만들기'라는 강의를 들은 후에,
강의에만 그치지 않고 응용하여 옷이 아닌 '주식'을 주문할 수 있는
증권사 MTS(Mobile Trading System App)을 클론코딩 해 봤습니다!

초기 홈 화면과 주문 화면에서 주식 수량을 체크해 버튼을 누르면
'체결' 팝업창이 화면에 뜨는 기능까지 구현해 봤습니다!
